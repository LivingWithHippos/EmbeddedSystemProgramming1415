installazione

Created Tuesday 28 April 2015

  1. Scaricare e installare l'Android Studio (il plugin per Eclipse non verrà più aggiornato nonostante possa essere leggermente più stabile) dall'indirizzo https://developer.android.com/sdk/index.html
  2. Scaricare e installare l'ultima versione di Java da http://www.oracle.com/technetwork/java/javase/downloads/jdk8-downloads-2133151.html
  3. Aprire l'Android SDK Manager e installare/aggiornare
      1. sotto Tools:
        • Android SDK Tools
        • Android SDK Platform-tools
5.sotto Android 5.0.1 (API 21):
  • SDK Platform
  • Google APIs
  • Arm EABI v7a system image
6.sotto ad Extras:
  • Android support library
  • Google USB driver

COLLEGAMENTO TELEFONO-PC

  1. verificare che sia attivato da impostazioni->opzioni sviluppatore->Debug Android
  2. collegare il telefono al computer, aspettare che si installino i driver, in caso di problemi conviene scaricare i driver ufficiali per il telefono dal sito della casa madre, installarli e riavviare il computer
  3. Il telefono una volta collegato al pc dovrebbe mostrare un messaggio riguardante l'autorizzazione ad usare il pc per il debug, accettare
  4. In caso di assenza telefono, o per simulare il nexus 7, si può utilizzare l'AVD Manager, incluso nella suite android studio:
    1. aprirlo ed andare su create
    2. scegliere nexus 7 2012 e dare ok

specifiche

Created Tuesday 28 April 2015

Le specifiche per il progetto sono:

  • DISPOSITIVO DI PROVA: Nexus 7 (si suppone versione 2012)
    • Versione Android 5.0 - 5.1
    • dimensione schermo 7 pollici
    • risoluzione 800*1280
  • DISPOSITIVI DISPONIBILI:
    • NEXUS 5 (Android 5.1, 5 pollici, 1920*1080)
    • MOTO G 2014 (Android 5.0.2, 5 pollici, 1280*720)
    • GALAXY S PLUS (Android 5.0.2, 4 pollici, 800*480)
  • STRUMENTI SVILUPPO:
    • Android Studio
  • API MINIME: 21 (Android 5.0)
